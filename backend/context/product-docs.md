# Product Docs

## Platform Overview

Softech Global Services provides ERP and AI-enabled SaaS services for business operations.

## Main Offerings

- ERP Software Services: ERPNext setup, module customization, migration, and support.
- AI MCP Integration with ERP: integrate AI and MCP-enabled context workflows into ERP processes.
- AI Automation Services: automate repetitive business tasks across finance, sales, support, and operations.
- AI Agent Solutions: deploy AI agents for customer service, internal helpdesk, and workflow orchestration.

## Typical Modules and Workstreams

- ERP workstreams: accounting, inventory, procurement, CRM, HR, and project operations.
- Automation workstreams: approvals, reminders, document workflows, and exception handling.
- Support workstreams: ticketing, customer messaging, knowledge base retrieval, and escalation routing.

## Supported Channels

- Web form
- WhatsApp
- Gmail

## Agent Operating Notes

- Search knowledge base first before making technical claims.
- Use customer history across previous interactions when available.
- Open tickets only for issues requiring formal tracking/follow-up.
- Escalate to human only for restricted, high-risk, or unresolved cases.
